Quick setup guide

Step 1: Paste into Google Apps Script ▼
  1. Go to script.google.com
  2. Click New Project
  3. Select all (Ctrl+A) and delete the existing code
  4. Paste the script you just copied
  5. Press Ctrl+S to save
Step 2: Fix permissions file (important!) ▼
  1. Click the ⚙️ Settings gear icon in the left sidebar
  2. Check "Show appsscript.json manifest file in editor"
  3. Click on appsscript.json in the file panel
  4. Replace all its contents with the JSON block included at the bottom of your script
  5. Save with Ctrl+S
Step 3: Run it ▼
  1. In the function dropdown, select startFresh
  2. Click ▶ Run
  3. Approve permissions when asked (click Advanced → Go to project)
  4. Wait for the log to say "You can now CLOSE THIS TAB"
  5. Close and sleep — albums appear in your Drive when done ✅
Troubleshooting ▼
Error 429 (quota exceeded): Enable billing at cloud.google.com/free. You won't be charged — free $300 credits apply.

Too many redirects: Use an incognito window (Ctrl+Shift+N) and sign in with only your wedding Google account.

0 photos found for an event: Folder name in the script must match Drive exactly — check spacing and capitalisation.

Script stuck: Run checkProgress to see how many photos are done, then resumeFromSaved to continue.
